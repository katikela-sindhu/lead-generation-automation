#  Lead Generation Automation Script

##  Overview

This project is a Python-based automation tool that collects and processes lead data from a public website.

---

##  Features

* Multi-page web scraping
* Data cleaning (remove duplicates, handle missing values)
* Automatic email generation
* Sorted and structured output
* Export to Excel

---

##  Tech Stack

* Python
* Requests
* BeautifulSoup
* Pandas
* OpenPyXL

---

##  Project Files

* `main.py` → Python script
* `leads.xlsx` → Output file
* `README.md` → Documentation

---

##  How to Run

pip install requests beautifulsoup4 pandas openpyxl
py main.py

---

##  Output

The script generates an Excel file containing:

* Full Name
* Description
* Email
* Location

---

##  Approach

* Scraped data using BeautifulSoup
* Processed data using Pandas
* Generated email addresses
* Cleaned and sorted the dataset
* Exported data to Excel
